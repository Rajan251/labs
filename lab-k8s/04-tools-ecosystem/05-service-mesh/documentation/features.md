# Service Mesh Features

## Overview
Istio provides a uniform way to secure, connect, and monitor microservices.

## Key Features

### Traffic Management
- **Routing**: Control traffic flow between services (canary, blue/green).
- **Resilience**: Retries, timeouts, circuit breakers, fault injection.

### Security
- **mTLS**: Automatic mutual TLS between services.
- **Authorization**: Fine-grained access control policies.

### Observability
- **Metrics**: Golden signals (latency, traffic, errors, saturation).
- **Tracing**: Distributed tracing via Jaeger/Zipkin.
- **Visualization**: Service topology visualization via Kiali.
