# Kubernetes Production-Grade Course

Welcome to the comprehensive Kubernetes Production-Grade Course repository. This repo contains labs, examples, and documentation to help you master Kubernetes operations.

## Repository Structure

*   **`labs/`**: Hands-on exercises organized by module.
    *   `01-getting-started/`: Deployments and Services.
    *   `03-ingress/`: Ingress controllers and resources.
    *   `04-statefulsets/`: Stateful applications with PVCs.
*   **`examples/`**: Advanced patterns and configurations.
    *   `helm-chart-example/`: Sample Helm chart structure.
    *   `operator-example/`: Custom Resource Definitions (CRDs).
*   **`docs/`**: Operational documentation and runbooks.
    *   `production-checklist.md`: 25+ item readiness checklist.
    *   `troubleshooting.md`: Cheat sheet for debugging clusters.
*   **`ci/`**: CI/CD pipeline examples.
    *   `github-actions/`: Workflow for building and pushing images.

## Getting Started

1.  Navigate to `labs/01-getting-started` and follow the README.
2.  Review `docs/production-checklist.md` before going to production.
